## Rules

### Git Commit Message Tags:
- `#feat`: Added some feature
- `#fix`: Fixed logical or programming bug
- `#design`: Changed design
- `#add`: Added non-feature files
- `#refactor`: Refactored code base including formatting
- `#del`: Deleted files
- `#chore`: Other
